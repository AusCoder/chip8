## Chip 8
This is a Chip 8 emulator written in C. It uses SDL2 draw the window and read keyboard events.

### SDL2
Installing on Mac OS using homebrew:
```
brew install sdl2
```
Paths to the header files and library files can be found using:
```
sdl2-config --cflags
sdl2-config --libs
```
